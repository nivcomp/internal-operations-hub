# Supplier Time Tracking

Supplier time tracking helps Yaniv understand delivery cost, supplier payment obligations, and hour bank usage.

## Time Entry Fields

- Project
- Supplier
- Date
- Hours
- Description
- Related scope item or task
- Status
- Agency review notes

## Time Entry Statuses

- Draft
- Submitted
- Approved
- Rejected

## Supplier View

Supplier can:

- See assigned work.
- Add time entries.
- View submitted and approved time.
- See amount owed for approved time.

Supplier cannot:

- See client price.
- See agency margin.
- See other suppliers' payments.

## Agency View

Yaniv can:

- Review submitted time.
- Approve or reject entries.
- Compare actual time against estimated supplier cost.
- See amount owed by supplier and project.
- Track hour bank usage when relevant.

## Paid Hours Rule

If a project uses a client hour bank, approved supplier time should reduce the available hours. Pending supplier time can be shown separately so Yaniv can anticipate overages.
