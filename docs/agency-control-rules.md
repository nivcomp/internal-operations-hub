# Agency Control Rules

The agency must stay in control of scope, price, supplier assignment, and client-facing commitments.

## Control Rules

- Yaniv controls final scope.
- Yaniv controls client price.
- Yaniv controls supplier cost assumptions.
- Yaniv controls supplier assignment.
- Yaniv controls what clients see.
- Yaniv controls what suppliers see.
- Yaniv decides when work is ready to start.

## Start Work Rules

Supplier work should not start until:

- Scope is approved, and
- Payment is received or paid hours are available, and
- Yaniv has assigned the supplier, and
- The project is marked ready for work.

## Change Request Rules

- Client requests are captured as requests, not immediate commitments.
- Yaniv reviews every change request.
- Changes must be priced before approval.
- Client must approve priced changes before supplier work begins.

## Visibility Rules

Client can see:

- Client-facing scope
- Project status
- Approval requests
- Payment or hour balance status
- Client-facing messages
- Client-visible files and links

Supplier can see:

- Assigned work
- Supplier-facing scope items
- Relevant files and links
- Delivery instructions
- Their own time entries and payment status

Only Yaniv can see:

- Client price and supplier cost together
- Margin
- Internal notes
- Supplier comparison
- All project controls

## AI Control Rule

AI can draft, summarize, classify, and suggest. AI should not approve scope, set final prices, assign suppliers, or send final commitments without Yaniv.
